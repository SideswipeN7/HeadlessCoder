HeadlessCoder lets you drive coding-agent CLIs on this machine from any device on your network.

  • Sessions — the left list shows every session. Group them Recent or By agent (agent groups collapse when you tap the header).
  • New session — pick an agent + working directory, then chat. Claude sessions resume; Gemini/Copilot run one-shot.
  • Share — 🔗 in the header copies a link that opens the current conversation.
  • Permission mode — Ask skips tools needing approval, Accept edits auto-approves file edits, Plan makes no changes, Bypass runs everything.
  • Send — Enter sends, Shift+Enter for a newline. Replies render as Markdown.
  • Working mode — Shift+Tab in the message box cycles Ask → Accept edits → Plan → Bypass.
  • In-private — start a session that stays out of the list; minimize it with ▁ and restore it from the pill in the corner.
  • Appearance — switch color scheme, light/dark, and hand-tune colors here. All saved in this browser.

CLI flags: --no-sleep, --pass, --no-pass, --free-style, --no-history, --port, --help.
